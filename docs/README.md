# 우테코 3주차 미션 Lotto!

## 구현 목록

1. [ ] 난수 로또 생성 클래스 생성
2. [ ] 유저 input 받는 클래스 생성
   1. [ ] 구입금액 입력 &rarr; 1000원 단위인지 확인
   2. [ ] 당첨번호 입력 &rarr; "," 로 구분된 6개 숫자
   3. [ ] 보너스 번호 입력 
   4. [ ] 각 숫자 유효성 검증
      1. [ ] 번호 개수 검증 &rarr; 첫 리스트 사이즈 체크
      2. [ ] 번호 크기 검증 &rarr; 1~45 사이
      3. [ ] 중복되지 않은 값 검증 &rarr; hashSet 사이즈 체크
      4. [ ] 보너스 숫자도 중복되지 않는지 체크 
3. [ ] 메인 로또 로직 
   1. [ ] 통계 로직
   2. [ ] 각 로또마다 일치하는 숫자 계산
